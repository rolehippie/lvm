## Default Variables

### lvm_group_volumes

List of volumes to manage per group

#### Default value

```YAML
lvm_group_volumes: '{{ lvm_volumes | default([]) }}'
```

#### Example usage

```YAML
lvm_group_volumes:
  - name: lv_data
    group: vg_data
    disks:
      - /dev/sda
    fstype: xfs
    size: 100g
    resizefs: True
    mountpoint: /mnt/data
    mountopts:
      - ro
      - noauto
    state: present
```

### lvm_host_volumes

List of volumes to manage per host

#### Default value

```YAML
lvm_host_volumes: []
```

#### Example usage
